Project Description

Use an array of hexadecimal values to randomly generate a background colour for the web page.

At the bottom of the index.html file, you can see a JavaScript function to generate a random number.

Creating the JavaScript code

Follow these steps:

  1. In the JavaScipt code at the bottom of index.html, create an array named arrHex to hold the 16 possible values 0-9 and A to F that make up a colour hex code.
  2. Get references to the button named btn-color that will trigger the colour change and the color-display web area that will display the current hex colour code.
  3. Add an event listener so that when the button is clicked, the background colour changes.
  4. Inside the event listener, create a new variable strHexColor that starts with the hash character # and uses a for loop to add six random characters from the arrHex array to the string.
  5. Update the text content of the color-display display area so the user can see the hex code.
  6. Update the style.backgroundColor property of the theme-js div with the new random colour.

Save your index.html file with the JavaScript code, view the web page in your browser and verify the button works correctly.
