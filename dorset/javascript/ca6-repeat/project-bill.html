Project Description

Create a basic script for calculating and displaying totals and sub-totals on a web page.

In the index.html file, you can see:

  • Three inputs, named num1, num2 and percentage
  • Two outputs, named subtotal and final-total.
  • And a button named calculate-btn

Creating the JavaScript code

Follow these steps:

  1. At the bottom of index.html, add JavaScript code to create references to the three three input elements, the two output elements, and the button.
  2. Create a function named calculateTotal().
  3. Inside this function, get the values from the two number inputs, and get the value from the percentage input.
  4. Convert these values from strings to floating point numbers. If any input is empty or invalid, treat it as zero.
  5. Add the two numbers together to get the subtotal.
  6. Calculate the percentage amount by multiplying the subtotal by the percentage and dividing by 100.
  7. Add the percentage amount to the subtotal to get the final total.
  8. Update the text inside the "Subtotal" display area with the subtotal, rounded to two decimal places.
  9. Do the same for the final total.
  10. Add an event listener to the button so that when it is clicked, your calculation function runs.

Save your index.html file with the JavaScript code, view the web page in your browser and verify your code runs correctly.
