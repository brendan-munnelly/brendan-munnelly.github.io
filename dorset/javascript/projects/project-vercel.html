Project Description

Uses the free version of the Vercel code generator and hosting platform to create a copy of the weather app built in the previous project.

You can view a completed version of the project here.

Creating and deploying the app

Follow the steps below:

  1. Got to v0.dev and register for the free version of the service.
  2. As far as is supported by the free version of Vercel, draft a prompt that resembles the functionality and features of the weather app you created earlier.
  3. Deploy your app to the Vercel hosting platform.
