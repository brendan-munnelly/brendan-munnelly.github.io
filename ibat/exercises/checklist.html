Introduction

This web page lists and explains the marking and grading scheme for the IBAT Web Design Diploma. The course has three main elements:

  • 13 Exercises
  • Six Web Design Projects
  • Personal Portfolio Website

The deadline for completion of all tasks is Thursday 30th May 2025.

Exercises

You will need to complete correctly nine exercises.

See a complete list of exercises.

Web Design Projects

You will need to complete correctly the following six web design projects:

  • The Snazzy Fashion project
  • The Smooth Smoothies project
  • The Crypto Wallet project
  • The Hero Block Gallery project
  • The Multi-column Gallery project
  • The Book store project

See a complete list of projects.

You can create these projects with the Lunevedy app.

Personal Portfolio Website

You will need to create a four-page personal portfolio website, using the Lunevedy app.

You will be expected to:

  • Add your own hero image, colour scheme, contact details, and logo.
  • Use at least one font from Google Fonts.
  • Add a link to your account on Google Analytics to all four web pages.
  • Add a 'pop-up' cookie privacy message to all four web pages.
  • Update your contact form with your unique Formspree ID.
  • Create a sitemap.xml file for your website and submit it with the Google Search Console.

Marking Scheme and Grades

The grading and marking scheme for the IBAT Web Design Diploma is as follows:

  • 70–100%: Distinction
  • 69–50%:   Merit
  • 49–40%:   Pass
  • 0–39%:     Fail

Exercises: Marks (40%)

Correctly completing all nine exercises earns a total mark of 40% (Pass grade).

Web Design Projects: Marks (15%)

Correctly completing all six web design projects earns a mark of 15%.

In other words, students who complete the exercises and web design projects will earn a mark of 55% (Merit grade).

Personal Website: Marks (45%)

Completing your personal website earns the balance of 45% in marks. These marks are sub-divided as follows:

  • Google Analytics, privacy 'pop-up' message and FormSpree ID: Adding these extra features earns another 15%. (55% + 15% = 70%, Distinction)
  • Personalise design: Applying your own images, colours, fonts, and other design features adds a further 30% to give a total of 100%.
