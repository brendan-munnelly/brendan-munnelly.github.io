Introduction

This Tutorial lists and explains the marking and grading scheme for the IBAT Web Design Diploma. The course has three main elements:

  • Six Exercises
  • Four Web Design Projects
  • Personal Portfolio Website

Exercises

You will need to complete the following exercises:

  • Six sample web pages, named page-1.html to page-6.html, and their CSS files and images.
  • Three sample hero block exercises, hero-text.html, hero-image.html and hero-video.html, and their CSS files and images.

See a complete list of exercises.

Web Design Projects

You will need to complete correctly the following four web design projects:

  • The Vincent Van Gogh project
  • The Snazzy Fashion project
  • The Smooth Smoothies project
  • The Crypto Wallet project

See a complete list of projects.

You can create these projects with the Lunevedy app.

Personal Portfolio Website

You will need to create a four-page personal portfolio website, using with the Lunevedy app.

Ensure you have customised your website in accordance with this pre-upload checklist.

After performing all these changes, you have two further updates you need to apply to your portfolio web pages:

  • Add a 'pop-up' cookie privacy message.
  • Add a link to your account on Google Analytics.
  • Update your contact form with your unique Formspree ID.

Marking Scheme and Grades

The grading and marking scheme for the IBAT Web Design Diploma is as follows:

  • 70–100%: Distinction
  • 69–50%:   Merit
  • 49–40%:   Pass
  • 0–39%:     Fail

Exercises: Marks (40%)

Correctly completing all nine exercises earns a total mark of 40% (Pass grade).

Web Design Projects: Marks (15%)

Correctly completing all four web design projects earns a mark of 15%.

In other words, students who complete the exercises and web design projects will earn a mark of 55% (Merit grade).

Personal Website: Marks (45%)

Completing your personal website earns the balance of 45% in marks. These marks are sub-divided as follows:

  • Google Analytics, privacy 'pop-up' message and FormSpree ID: Adding these two extra features earns another 15%. (55% + 15% = 70%, Distinction)
  • Personalise design: Applying your own images, colours, fonts and other design features adds a further 30% to give a total of 100%.
